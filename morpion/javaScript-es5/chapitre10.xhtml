Chapitre 10: Modifier les feuilles de style

CSS

Les styles d'un élément peuvent être directement définis dans son attribut style ou peuvent appartenir à une feuille de style en utilisant les sélecteurs. Il est possible de modifier les styles de la feuille de style ou de l'attribut.

Modifier un style en direct

Propriété style

La propriété style d'un élément correspond à la valeur de son attribut mais ne correspond pas aux propriétés CSS qui lui sont définies dans une feuille de style. Soit le code suivant.

<html>
    <head>
        <style>
            h1 {
                color: blue;
            }
            .green {
                color: green;
            }
        </style>
        <script type="text/javascript" defer src="script.js"></script>
    </head>
    <body>
        <h1 id="titre">Titre</h1>
    </body>
</html>

Les attributs d'un élément sont accessibles via la méthode getAttribute ou par l'accesseur de propriété comme étudié au chapitre précédent. Bien que le titre ait une propriété color définie dans la balise style, la propriété style de l'élément HTML ne contient pas de propriété color car la balise ne comporte pas d'attribut style.

Propriété style // ""
console.log(document.getElementById("titre").style.color);

En utilisant la propriété style d'un élément c'est bien à son attribut style qu'une valeur est affectée. Bien que la couleur du titre ait été définie dans la feuille de style, en spécifiant une couleur dans son attribut style elle devient prioritaire.

var titre = document.getElementById("titre");
    titre.style.color = "red";

// red
console.log(titre.style.color);

En modifiant l'attribut classe de l'élément pour que lui soit affecté la couleur correspondante au sélecteur de cette classe, l'élément conserve la couleur de son attribut qui est prioritaire sur la feuille de style.

var titre = document.getElementById("titre");
    titre.style.color = "red";

titre.className = "green";
camelCase

Le nommage des propriétés de la propriété style d'un élément HTML doit se faire en camelCase, ainsi background-color doit s’écrire backgroundColor, z-index devient zIndex etc.

Propriété composé var titre = document.getElementById("titre");
    titre.style.backgroundColor = "yellow";

Modifier une feuille de style

Feuille de style

Utiliser une feuille de style et opérer les modifications dans la feuille est une meilleur pratique que de modifier la valeur de l'attribut style.

getComputedStyle

La méthode getComputedStyle de l'objet window sert à récupérer les styles présents dans les feuilles de style sous forme d'objet CSSStyleDeclaration. Le premier paramètre correspond à un nœud de type élément, le second paramètre facultatif est une chaine de caractères pour spécifier son pseudo élément. La méthode lance une erreur TypeError si le premier paramètre n'est pas un élément HTML.

var CSSStyleDeclaration = window.getComputedStyle(element[, pseudo]);
getComputedStyle var titre = document.getElementById("titre");
    
// CSS2Properties
console.log(window.getComputedStyle(titre));

getComputedStyle

Cette méthode n'est pas prise en charge pour les anciennes versions d'Internet Explorer comme la version 8. Pour utiliser cette méthode sans problème de compatibilité il faudra vérifier sa présence au sein du navigateur comme dans l'exemple çi-dessous.

var titre = document.getElementById("titre"),
    style;

if (window.getComputedStyle) {
    style = window.getComputedStyle(titre);
} else {
    style = titre.currentStyle;
}
    
// CSS2Properties
console.log(style);
getPropertyValue

La méthode getPropertyValuede l'objet CSSStyleDeclaration renvoie le style correspondant à la propriété fournie en paramètre sous forme de chaine de caractères.

var propriete = CSSStyleDeclaration.getPropertyValue(nom);
getPropertyValue var style = window.getComputedStyle(document.getElementById("titre"));

// rgb(0, 0, 255)
console.log(style.getPropertyValue("color"));
insertRule

Une balise style contient l'objet CSSStyleSheet, cet objet possède des méthodes pour insérer des sélecteurs et leurs règles. Le premier argument représente le sélecteur et sa règle, le second est un entier pour sa position dans la feuille. Cette méthode renvoie la position où la règle est insérée.

CSSStyleSheet.insertRule(regle, position);
insertRule var style = document.getElementsByTagName("style")[0].sheet;

// 0
console.log(style.insertRule("h1 { background: yellow; }", 0));
deleteRule

La méthode deleteRule supprime une règle de la feuille de style, son paramètre correspond à la position de la règle dans la feuille de style. La méthode renvoie undefined

CSSStyleSheet.deleteRule(position);

La propriété cssRules d'un objet CSSStyleSheet contient l'ensemble des règles CSS, il possède une taille et une propriété selectorText qui renvoie le sélecteur correspondant à la règle. Il est donc possible de le parcourir pour récupérer la position d'une règle.

deleteRule var style = document.getElementsByTagName("style")[0].sheet;
    
for (var i = 0, length = style.cssRules.length ; i < length; i++) {
    if ("h1" === style.cssRules[i].selectorText) {
        // undefined
        console.log(style.deleteRule(i));
        --length;
    }
}
removeProperty

La méthode removeProperty de l'objet CSSStyleDeclaration supprime une propriété de la règle en cours. La méthode renvoie soit la valeur de la propriété supprimée soit une chaine vide.

var propriete = CSSStyleDeclaration.removeProperty(nom);
removeProperty var style = document.getElementsByTagName("style")[0].sheet;
    
for (var i = 0, length = style.cssRules.length ; i < length; i++) {
    if ("h1" === style.cssRules[i].selectorText
     && style.cssRules[i].style["color"]) {
        // blue
        console.log(style.cssRules[i].style.removeProperty("color"));
        break;
    }
}
setProperty

La méthode setProperty de l'objet CSSStyleDeclaration ajoute une propriété à la règle en cours. La méthode renvoie undefined. Les méthodes d'écriture de l'objet CSSStyleDeclaration lève une erreur de type NoModificationAllowedError si le style est en lecture seule.

CSSStyleDeclaration.setProperty(nom, valeur [,important]);
setProperty var style = document.getElementsByTagName("style")[0].sheet;
    
for (var i = 0, length = style.cssRules.length ; i < length; i++) {
    if ("h1" === style.cssRules[i].selectorText) {
        style.cssRules[i].style.setProperty("color", "white");
        break;
    }
}
