Chapitre 08: Accéder aux nœuds du DOM

Atteindre les nœuds

L'objet document et ses éléments nous offrent des méthodes pour accéder à leurs noeuds. Ce chapitre expose les méthodes pour cibler un élément particulier, puis une collection d'éléments avant de voir les méthodes pour naviguer entre les nœuds.

Accéder à un élément

Document

Le corps du document peut s'atteindre en utilisant la propriété body du document. Que la balise body soit présente ou non cette propriété renvoie la représentation du corps du document.

var body = document.body;
defer <html>
    <head>
        <script type="text/javascript" defer src="script.js"></script>
    </head>
    <body>
        <ul id="identifiant"></ul>
    </body>
</html>

Nous utilisons un script distant qui possède l'attribut defer. Defer indique au navigateur que l’exécution du script doit se faire une fois l'ensemble document chargé, en son absence nous ne pouvons pas interagir avec ses éléments car il ne sont pas chargés et donc pas présents lors de l’exécution du script. Une fois que les évènement auront été introduits il ne sera plus utile d'utiliser defer qui pose des problèmes de compatibilité avec les vielles versions d'internet explorer.

document.body // <body>
console.log(document.body);
getElementById

La méthode getElementById renvoie soit l'élément unique identifié par son attribut id transmis en paramètre soit null.

var noeud = document.getElementById(id);
document.getElementById // <ul id="identifiant">
console.log(document.getElementById("identifiant"));
querySelector

La méthode querySelector renvoie soit l'élément identifié par un selector soit null.

var noeud = document.querySelector(selector);
document.querySelector // <ul id="identifiant">
console.log(document.querySelector("#identifiant"));
querySelector

La méthode repose sur la Selectors API et le navigateur doit prendre en charge la version du selector spécifié pour son bon fonctionnement. Indépendamment des selectors, cette méthode n'est disponible qu'à partir d'Internet Explorer 8 ou Opera 10 par exemple.

elementFromPoint

La méthode elementFromPoint renvoie soit l'élément pointé graphiquement par un couple de coordonnées x et y soit null.

var noeud = document.elementFromPoint(x,y);
document.elementFromPoint // <html>
console.log(document.elementFromPoint(0,0));

Accéder à une collection

HTMLCollection et NodeList

Des méthodes de l'objet document renvoient une collection d'éléments, nous complétons le fichier HTML déjà utilisé afin d'interagir avec.

<html>
    <head>
        <script type="text/javascript" defer src="script.js"></script>
    </head>
    <body>
        <ul id="identifiant" class="classe">
            <li name="nom"></li>
        </ul>
        <ul class="classe">
            <li name="nom"></li>
        </ul>
    </body>
</html>
getElementsByTagName

La méthode getElementsByTagName renvoie soit une collection d'éléments correspondants au tag spécifié en paramètre pour l'élément parent utilisé soit un tableau vide.

var collection = element.getElementsByTagName(tag);
getElementsByTagName // HTMLCollection[li, li]
console.log(document.getElementsByTagName("li"));

// HTMLCollection[li]
console.log(document.getElementById("identifiant").getElementsByTagName("li"));

Le nom du tag est représenté en capitale pour un document HTML et en minuscule pour un document XHTML ou XML.

getElementsByClassName

La méthode getElementsByClassName renvoie soit une collection d'éléments correspondants à la classe spécifiée en paramètre pour l'élément parent utilisé soit un tableau vide.

var collection = element.getElementsByClassName(className);
getElementsByClass // HTMLCollection[ul#identifiant.classe, ul.classe]
console.log(document.getElementsByClassName("classe"));
getElementsByClassName

La méthode getElementsByClassName n'est prise en charge pour le navigateur de Microsoft qu'à partir de la version 9 de Internet Explorer.

querySelectorAll

La méthode querySelectorAll renvoie soit une collection d'éléments correspondants aux sélecteurs spécifiés en paramètre pour l'élément parent utilisé soit un tableau vide.

var collection = element.querySelectorAll(selector);
querySelectorAll //  NodeList[ul#identifiant.classe, li, ul.classe, li]
console.log(document.querySelectorAll("ul, li"));
getElementsByName

La méthode getElementsByName renvoie soit une collection d'éléments correspondants au nom spécifié en paramètre pour l'élément parent utilisé soit un tableau vide.

var collection = element.getElementsByName(nom);
getElementsByName //  NodeList[li, li]
console.log(document.getElementsByName("nom"));

Naviguer entre les noeuds

Navigation

A partir d'un nœud il est possible de se déplacer dans le DOM et d'atteindre des nœuds qui ne sont pas de type élément.

firstChild

La propriété firstChild d'un nœud renvoie le premier enfant de ce nœud ou undefined, un enfant est un nœud. Les propriétés nodeName et nodeType renvoient le nom et le type de nœud.

var noeud = noeud.firstChild;
firstChild var noeud = document.getElementById("identifiant").firstChild;

// <TextNode textContent="\n            ">
console.log(noeud);
// #text
console.log(noeud.nodeName);
// 3
console.log(noeud.nodeType);
lastChild

La propriété lastChild d'un nœud renvoie le dernier enfant de ce nœud ou undefined.

var noeud = noeud.lastChild;
lastChild // <body>
console.log(document.firstChild.lastChild);
nextSibling

La propriété nextSibling d'un nœud renvoie le nœud suivant ou undefined.

var noeud = noeud.nextSibling;
nextSibling // <TextNode textContent="\n        ">
console.log(document.getElementById("identifiant").nextSibling);
previousSibling

La propriété previousSibling d'un nœud renvoie le nœud précédent ou undefined.

var noeud = noeud.previousSibling;
previousSibling // <ul id="identifiant" class="classe">
console.log(document.getElementById("identifiant").nextSibling.previousSibling);
parentNode

La propriété parentNode le nœud parent ou null si la propriété est appelée depuis le document..

var noeud = noeud.parentnode;
parentNode // <body>
 console.log(document.getElementById("identifiant").parentNode);

// <html>
 console.log(document.body.parentNode);
childNodes

La propriété childNodes renvoie la collection des nœuds enfants ou undefined.

var collection = noeud.childNodes;
childNodes // NodeList[<TextNode textContent="\n            ">, li, <TextNode textContent="\n        ">]
 console.log(document.getElementById("identifiant").childNodes);
